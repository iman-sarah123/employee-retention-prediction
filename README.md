# Employee Retention Prediction

## Overview
This project aims to predict employee attrition using Machine Learning techniques. The model analyzes employee-related factors such as job satisfaction, work environment, salary, and performance to identify employees who are likely to leave the organization.

## Objectives
- Predict employee attrition.
- Identify key factors influencing employee retention.
- Support HR teams in making data-driven decisions.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Techniques
- Data Preprocessing
- Feature Engineering
- Classification Models
- Model Evaluation

## Results
Built a predictive model capable of identifying employees at risk of attrition, helping organizations improve retention strategies.
